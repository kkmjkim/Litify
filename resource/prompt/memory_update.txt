[지시]
너는 주어진 [직전 발화 내역]에 나타나 있는 AI의 질문과 User의 답변을 토대로 [필요한 정보 목록]을 채우는 역할을 해야해. [필요한 정보 목록]은 key-value 쌍으로 이루어진 json 데이터이고, [직전 발화 내역]에 나타나 있는 AI의 질문은 [필요한 정보 목록]에서 value가 비어있는 key에 대한 질문 중 하나야. [직전 발화 내역]에 AI의 질문에 해당되는 key를 찾고, 그 key의 value 값에 User의 답변을 바탕으로 "" 부분을 채워넣으면 돼. User가 언급하지 않은 내용은 상상해서 작성하지 않도록 주의해.

1. User가 AI의 질문에 대해 모른다고 해도 그것 또한 정보를 제공한 것이기 때문에 사용자의 발화를 value 값으로 채워넣으면 돼.
2. 만약 여러 개의 key에 대한 답변을 알 수 있는 경우, 여러 개의 value를 채워넣어도 돼.
3. 만약 [필요한 정보 목록]의 모든 value들이 빈 문자열이 아닌 다른 문자로 채워졌다면 "finished": "false"를 "finished": "true"로 바꿔줘. 하지만 value가 “”인 key가 하나라도 있다면 "finished": "false"를 유지해야 해.
4. 이때, [출력]은 기존의 [필요한 정보 목록] 형태처럼 key와 value는 모두 큰따옴표로 묶인 json 형태로 출력이 되어야 해.

[직전 발화 내역]
AI: {previous_question}
User: {utterance}

[필요한 정보 목록]
{memory}
